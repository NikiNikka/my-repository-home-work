Команды при работе с терминалом

При работе с терминалом мы использовали определенные команды.

Команда git init
создает новый репозиторий Git.
Команда git status
показывает состояние рабочего каталога и раздела проиндексированных файлов.
Команда git add .
добавляет изменение из рабочего каталога в раздел проиндексированных файлов.
Команда git commit -m "#"
регистрирует изменения в Git.
Команда git push
используется для выгрузки содержимого локального репозитория в удаленный репозиторий, т.е. позволяет передать коммиты.
Команда git pull
используется для извлечения и загрузки обновлений из удаленного репозитория на локальный репозиторий.
